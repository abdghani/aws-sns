APIS

SMSANY lets you send sms to any where in the world within seconds . Just send a post request to the server with valid credentials and your message will be delivered. Obtain an access key by submitting your credentials and you are good to go.

REQUEST 

   POST : https://smsany.greyphase.xyz/api/sms/sendsms
   HEADER  :  Content-Type : application/json
   BODY :
        {
            phoneNumber : xxxxxxxxxx                    // target phone number
            message : (message you want to send)        // message BODY
            accessKey : valid access key                // valid access key 
        }

RESPONSE BODY

FAILURE

    RESPONSE BODY :
    {
        status : 'failure',                     
        reason : 'reason of failure' ,                
        params : 'request body sent',
    }

SUCCESS

    RESPONSE BODY :
    {
        status : 'success',                
        info : messageId,       //unique message Id on successfull delivery of message
    }
