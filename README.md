# SMSANY
SMSANY is sms service based on aws-sns. 

##Description
> - Generate API Key By simple authentication
> - Send Message to any Body over the Globe using the key
> - Easy Api Integration For your any Web Application
 
##Installaion
> - cd smsany
> -  npm install 
> -  bower install
> -  nodemon
